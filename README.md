# Excel Sales Dashboard Project

## Overview
This project is an interactive Excel dashboard used to analyze sales performance of sales executives across different regions.

## Features
- Total Sales Analysis
- Target Achievement Tracking
- Sales Executive Performance Monitoring
- Region-wise Sales Analysis
- KPI Dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

## Dataset
The dataset contains:
- Employee Code
- Sales Executive Name
- Region
- Daily Sales
- Total Sales
- Sales Target
- Target Achievement
